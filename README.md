# compile
```
git clone https://github.com/MESYETI/yedit
cd yedit
mkdir bin
make
```
# run
```
./bin/yedit
```
# keybinds
Control Q : quit<br>
Control S : save file<br>
Control G : Enable experimental syntax highlighting
Control H : Show help menu
